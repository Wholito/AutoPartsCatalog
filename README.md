# Каталог автомобильных запчастей и аксессуаров

Мобильное приложение на Java для Android — каталог запчастей с CRUD-операциями.

## Требования и реализация

1. **Пользовательский интерфейс:**
   - Главный экран — список запчастей
   - Экран детали — просмотр/редактирование запчасти (заголовок, описание, дата)
   - Экран настроек — выбор темы

2. **Сплэш-экран:** показывается при запуске в течение ~2 секунд.

3. **Локализация:** русский и английский (`values/strings.xml`, `values-ru/strings.xml`).

4. **Тема интерфейса:** светлая, тёмная и системная тема; выбор сохраняется через `SharedPreferences`.

5. **Локальная БД (SQLite):**
   - Создание, редактирование и удаление записей
   - Поля: заголовок, описание, дата

## Сборка

```bash
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`

## Запуск

Подключите устройство или эмулятор и выполните:

```bash
./gradlew installDebug
```
